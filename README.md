# AS-POO

Projeto desenvolvido em Java utilizando Spring Boot e Gradle para a disciplina de Programação Orientada a Objetos.

## Autores

- **Pedro Afonso**
- **José Eduardo**

## Tecnologias

- Java
- Spring Boot
- Gradle
- RESTful API

## Pré-requisitos

Antes de começar, você precisará ter instalado em sua máquina:

- [Java JDK 17+](https://www.oracle.com/java/technologies/downloads/)
- [Git](https://git-scm.com/)
- [Postman](https://www.postman.com/downloads/) (para testar a API)

## Como clonar e executar o projeto

### 1. Clonar o repositório

```bash
git clone https://github.com/Jose-Edu/AS-POO.git
cd AS-POO
```
### 2. Executar a aplicação

Após clonar o projeto, você pode executar a aplicação usando o Gradle:

```bash
./gradlew bootRun
```

Ou, caso esteja no Windows:

```bash
gradlew.bat bootRun
```

A aplicação será iniciada em:

http://localhost:8080
